# 🎯 Quiz Game

A simple and interactive quiz application built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

* Multiple choice questions
* Real-time score tracking
* Progress bar indicator
* Instant feedback (correct / incorrect answers)
* Final result with performance message
* Restart quiz functionality

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 📖 How It Works

1. Click the **Start Quiz** button
2. Answer each question by selecting one option
3. Get instant feedback after each answer
4. Track your progress and score
5. View your final result at the end
6. Restart the quiz anytime

---

## 🎯 Purpose

This project was built to practice:

* DOM manipulation
* Event handling
* Dynamic UI updates using JavaScript
